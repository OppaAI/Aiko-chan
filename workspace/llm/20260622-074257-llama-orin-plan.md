Install CUDA 12.2 and cuDNN 9.0 for Jetson Orin compatibility. Verify with `nvcc --version` and `cat /usr/include/cudnn_version.h`.

Clone llama.cpp repository and build from source with `--enable-cuda`.

Install Python dependencies: PyTorch (CUDA) + torchvision + torchaudio.

Download Ministral3-3B-instruct model weights from HuggingFace Hub or official source.

Configure llama.cpp to load model with `--weights Ministral3-3B-instruct.safetensors` or `.bin`.

Run inference with `--tokenizer Ministral3-3B-instruct.tokenizer` and `--max_tokens 100` for testing.

Check GPU memory usage with `nvidia-smi` during execution.
